# Auth Single Sign On (SSO) with JSON Web Token (JWT), Spring Boot - Framework

## What you'll need
- JDK 1.8+
- Maven 3+

## Stack
- Java
- Spring Boot
- FreeMarker

## Run
- Run [Authentication Service](sso-jwt-auth): `mvn spring-boot:run`
- Run Resource Service 1: `mvn spring-boot:run -Dserver.port=8081`
